# Oblig1 DAT154 - Traffic Light Intersection Simulator

A Windows desktop application that simulates a traffic light-controlled intersection with animated vehicles.

The simulation displays two perpendicular roads with traffic lights that cycle through red, red/yellow, green, and yellow states. Vehicles spawn on each road at configurable probabilities and queue at the intersection, waiting for the light to turn green before proceeding. Spawn probabilities can be adjusted in real-time using the arrow keys or through a dialog box.
